# autolibrary
Школьный проект за 11 класс

План/особенности проекта
1. БД PostgreSQL
2. ORM SQLAlchemy
3. Telegram-бот для взаимодействия с пользователями
4. Сайт FastAPI для взаимодействия с БД

venv: Python 3.10.0, PostgreSQL 14.4(5)
